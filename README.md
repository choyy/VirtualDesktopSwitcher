# Virtual Desktop Switcher

Windows 11 虚拟桌面快速切换工具，使用 `Alt + 1~9` 快捷键一键切换虚拟桌面，带有桌面状态指示器和丰富的自定义选项。

## 功能特性

- 使用 `Alt + 1` 切换到桌面1
- 使用 `Alt + 2` 切换到桌面2
- 使用 `Alt + 3` 切换到桌面3
- ...以此类推，支持 `Alt + 1` 到 `Alt + 9`
- 显示桌面状态指示器，可自定义位置、大小和样式

## 编译方法

### 环境要求
- Windows 11
- Visual Studio 2022+（含 MSVC C++ 工具链）
- [xmake](https://xmake.io/) 构建工具

### 编译

```bash
xmake

# 生成 Visual Studio 项目
xmake project -k vsxmake
```

## 使用方法

1. 运行 `VirtualDesktopSwitcher.exe`
2. 系统托盘出现图标，屏幕中显示桌面状态指示器，可在托盘图标设置指示器的位置和样式
3. 使用 `Alt + 1~9` 切换虚拟桌面
4. 右键托盘图标进行设置和管理
5. 双击托盘图标切换开机自启动

所有设置自动保存到 `%APPDATA%\VirtualDesktopSwitcher\settings.ini`。

## 常见问题

- 切换虚拟桌面时，任务栏图标闪烁问题

  可通过在设置中禁止任务栏闪烁来解决，具体步骤如下：设置->个性化->任务栏->任务栏行为->取消勾选"显示任务栏应用上的闪烁"。**但这样做的问题是程序正常的闪烁（如应用消息提示）也会被禁止**。

- 让任务栏显示所有虚拟桌面打开的窗口图标

  打开“设置”，选择“系统” > “多任务处理” > “桌面” > “在任务栏上，显示所有打开的窗口”，在下拉菜单中，将默认的“仅在我正在使用的桌面上”改为“在所有桌面上”
